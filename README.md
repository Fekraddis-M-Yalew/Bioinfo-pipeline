# Bioinformatics Pipeline       
## Bioinformatics analysis pipeline for assignments.


## Project Description
This project is a simple bioinformatics pipeline that performs quality control (QC) on FASTQ files and counts variants from VCF files.

## Dependencies
The following tools and packages are required:
- Python 3.14
- gzip (Python module)
- os (Python module)

## Installation
To install dependencies:
- Install Python 3.14 from https://www.python.org
- No additional packages are required as gzip and os are built-in

## Execution
To run the pipeline:

```bash
python src/pipeline.py
